    Adding textures & custom models

    Three.js powered titles share the loadThree helper found in games/utils/three-loader.js. After resolving the module you can load textures or GLTF models using standard loaders: 

    const THREE = await loadThree();
const textureLoader = new THREE.TextureLoader();
const metalPanel = await textureLoader.loadAsync('assets/textures/panel.png');
const material = new THREE.MeshStandardMaterial({ map: metalPanel, metalness: 0.7 });
const panel = new THREE.Mesh(new THREE.BoxGeometry(2, 2, 0.1), material);
scene.add(panel);

    Place texture assets inside assets/textures (create the folder if it does not exist) and serve them relative to the project root. For GLTF files, use THREE.GLTFLoader from the examples/jsm/loaders bundle and reuse the same asynchronous pattern.

    Expanding narrative & scoring

    Each module exposes a startGame function that receives container, theme, and onComplete. Use this hook to add story beats, wave systems, or bonus challenges. Persist shared utilities in games/utils so the arcade shell can keep the experience consistent across titles.
